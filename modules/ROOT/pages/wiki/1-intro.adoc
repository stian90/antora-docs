== Markeringsspråk
Nordkontakt benytter Asciidoc som markeringsspråk i dokumentasjon

== Eksport til pdf

=== Docker 
[source,console]
---
docker run --rm \
-v ./:/document madduci/docker-asciidoctor-pdf \
-a pdf-themesdir=theme -a pdf-theme=nk \
-a pdf-fontsdir=fonts \
-a pdf-fontsdir=/document/resources/fonts /document/book.adoc
---

=== Ved asciidoctor installert

[source,console]
---
  asciidoctor -r asciidoctor-pdf -b pdf \
  -a pdf-theme=nk \
  -a pdf-themesdir=theme \
  -o book.pdf book.adoc
---
